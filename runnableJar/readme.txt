
﻿							  Магазин покупок

						----------Стартовая панель----------

Работа программы начинается с двойного клика, в таком случае автоматически будет запуск с Derby. 
Так же можно запустить через консоль, например: java -jar ShopOfItemsNew.jar mysql, будет запуск MySQL.
Derby - для работы с этой базой данных будет создан лог файл и папка с файлами(derby.lod, ShopOfItemsDB) рядом с запускаемым Jar файлом.
MySQL - для работы с этой базой данных нужно что бы был установлен сервер MySQL. 


					----------Панель успешного соединения с базой данных----------


Будет надпись успешного соединения с базой данных.


						----------Просмотр товаров---------- 


Предоставляет лист всех товаров


					       ----------Просмотр покупателей----------


Предоставляет лист всех покупателей


						----------Просмотр покупок----------


Предоставляет лист всех покупок


						 ----------Добавить товар----------


Предоставляет возможность добавить товар(Имя, цена, количество)


					       ----------Добавить покупателя----------


Предоставляет возможность добавить покупателей(Имя, Фамилия, возраст, пол, адресс, телефонный номер)


						 ----------Сделать покупку----------


Предоставляет возможность сделать покупку(Имя товара, количество товара, имя покупателя)


					         ----------Изменить товар----------


Предоставляет возможность изменить товар(Имя, цена, количество)


					       ----------Изменить покупателя----------


Предоставляет возможность изменить покупателя(Имя, Фамилия, возраст, пол, адресс, телефонный номер)

					            
					             ----------Меню бар----------


Панель Товары:
Просмотр товаров, Добавить товар, Изменить товар.
Панель Покупатели: 
Просмотр покупателей, Добавить покупателя, Изменить покупателя.
Панель Покупки:
Просмотр покупок, Сделать покупку.
